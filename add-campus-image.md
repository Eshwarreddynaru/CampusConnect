# How to Add Campus Background Image

## Quick Steps:
1. Right-click on the campus image in the chat
2. Select "Save image as..." or "Save picture as..."
3. Navigate to your project folder: `kare_lost_found_v2/public/`
4. Name the file: `campus-background.jpg`
5. Click Save
6. Refresh your browser

## File Location:
The image should be saved at:
```
kare_lost_found_v2/public/campus-background.jpg
```

## Verification:
After saving, you should see the campus image as the hero background with mountains and university buildings.

## Troubleshooting:
- Make sure the filename is exactly: `campus-background.jpg`
- Make sure it's in the `public` folder (not `src` or any other folder)
- Try refreshing the browser (Ctrl+F5 or Cmd+Shift+R)
- Check browser developer tools for any 404 errors

The code is already configured and waiting for this image file!
